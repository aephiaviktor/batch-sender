BATCH SENDER — WINDOWS PORTABLE

1. Extract the entire ZIP to a normal folder. Do not run it from inside the ZIP.
2. Double-click create-desktop-shortcut.vbs once.
3. Start Batch Sender from the new desktop shortcut, or double-click launch-batch-sender.vbs.
4. Open Wallet Settings and enter your own RPC/Aephia details.
5. Use the Update button inside the app for future versions.

Wallet settings and encrypted secrets are stored outside this application folder and are not included in the ZIP.
